# Typeset

Use this flow when the interface reads as generic, muddy, inconsistent, or tiring to scan.

## Preparation

1. Load `impeccable` first.
2. If design context is missing, run `impeccable teach`.
3. Review [`../impeccable/reference/typography.md`](../impeccable/reference/typography.md).

## What to improve

- font choice relative to brand voice,
- hierarchy between headings, labels, body, and metadata,
- size and weight contrast,
- line length and line height,
- consistency of text roles across the surface.

## Rules

- Do not default to the usual safe fonts.
- Do not fix hierarchy only with color.
- Do not let long passages run without width control.
- Keep changes systematic so the same text role looks the same everywhere.
