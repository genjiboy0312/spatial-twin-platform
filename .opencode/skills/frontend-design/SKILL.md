# Frontend Design

This deprecated upstream wrapper is included for completeness only within the local `impeccable` family.

Redirect to `impeccable` instead. In this local bundle:
- `frontend-web` or `mobile-app` stay the primary implementation packs,
- `impeccable` provides the companion design direction and anti-slop layer,
- and this wrapper is not a primary routing choice.

Keep `frontend-design` present for completeness only. Do not perform design work from `frontend-design`; treat it as a redirect to `impeccable`.
