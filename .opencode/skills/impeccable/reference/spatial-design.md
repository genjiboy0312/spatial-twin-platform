# Spatial design reference

Use this reference when the problem is layout rhythm, spacing, grouping, or composition.

## Core rules

- Use a coherent spacing scale and semantic tokens.
- Prefer `gap` for sibling rhythm.
- Group related elements tightly and separate unrelated ones generously.
- Break the grid intentionally only where emphasis or storytelling benefits.

## Composition guidance

- Give the eye a clear first stop.
- Let one strong gesture carry the page instead of repeating the same card pattern everywhere.
- Keep dense product UI structured and predictable; keep expressive marketing surfaces composed, not chaotic.

## Anti-slop watchouts

- Cards nested inside cards.
- Every section using the same padding and the same rhythm.
- Metric tiles repeated as the only layout idea.
- Centering everything when the content wants reading flow.
